## Alcance

Esta carpeta no contiene aun el informe final de Unidad 6. Tampoco reemplaza los GitHub Issues ni registra defectos reales.

La gestion formal de defectos se realizara despues, usando evidencias verificables y trazabilidad contra pruebas, commits, workflows o documentacion.

## Criterios de trazabilidad futura

Todo defecto debera relacionarse con al menos una fuente verificable:

- Prueba automatizada o manual.
- Evidencia de ejecucion.
- Commit o diff relacionado.
- Workflow de GitHub Actions.
- Documentacion tecnica o Wiki.
- Issue de seguimiento, cuando exista.
